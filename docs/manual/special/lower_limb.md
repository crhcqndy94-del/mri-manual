# 下肢血管MRA

> このページは準備中です。内容は順次追加されます。

---

## 概要

下肢血管MRAは、大動脈〜下肢末梢動脈の狭窄・閉塞・動脈瘤などを非侵襲的に評価する検査です。

主に閉塞性動脈硬化症（ASO）、バージャー病、動脈瘤などの評価に使用されます。

---

## 撮像範囲

- 腹部大動脈〜腸骨動脈
- 大腿動脈
- 膝窩動脈〜下腿動脈（必要に応じて）

---

## 撮像方法

| 方法 | 特徴 |
|------|------|
| 造影3D-MRA | 最も一般的。ガドリニウム使用 |
| Time-SLIP（非造影） | 腎機能障害例に有用 |
| TOF MRA | 小範囲の評価に使用 |
