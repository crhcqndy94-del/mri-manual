- ヘルニアがあれば**その辺りを連続スライス**で撮像
- 圧迫骨折・メタ疑い：TRAは椎間板ではなく**椎体を連続撮像**
- **メタ疑い：DWIBSを撮像**
- 多発性骨髄腫：**DWIBSとT1mDixon CORを撮像**
:::
