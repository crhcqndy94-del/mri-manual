# 頭部MRI 脳腫瘍

最終改訂：2025年11月4日

## ■ 検査概要

## ■ 断面設定（造影）

### CE 3D TRA
- 範囲：鼻根部から頭頂部まで撮りきる（角度はつけない）
- 撮像後：鼻根部と橋下縁のラインに合わせてMPR（1mm 170枚）
- MPR COR・SAGは必要であれば適宜作成

:::warning
- CE 3D TRA は角度をつけない
- T2*Wは何かあれば追加して撮像する
:::
