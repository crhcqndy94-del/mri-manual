:::info 運用

- FOV・スライス厚・枚数は頭部の大きさで適宜変更する
- 小児無熱性けいれん → FLAIR COR（海馬に垂直）を追加撮像
- 小児科寝かせて撮像する場合は16時枠を2単位押さえる（自動で2単位枠）
- 眠剤使用：検査2時間前に来院
- 真空固定具使用：同意書 + 小児科医師立ち会い
- 体動 → Motion Free（MF）またはRapid を使用
:::
