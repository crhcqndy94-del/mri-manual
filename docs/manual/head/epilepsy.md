# 頭部MRI てんかん・認知症・アルツハイマー・パーキンソン

最終改訂：2025年11月4日

## ■ 検査概要

てんかん・認知症・アルツハイマー病・パーキンソン病に対する検査（Brain epilepsy / Alzheimer）。
海馬評価のための 3D FLAIR COR が特徴的。

---
