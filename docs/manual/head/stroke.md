# 頭部MRI 脳卒中

最終改訂：2025年11月4日

## ■ 検査概要

脳梗塞・脳出血・TIA（一過性脳虚血発作）が疑われる症例に対して施行する。
DWI・FLAIR・T2*Wを中心に、MRAで血管評価を行う。
緊急オーダーには Smart Brain を第一選択とし、対応不可の場合は Rapid を使用する。

:::warning 緊急対応
緊急オーダーの第一選択は Smart Brain。Rapidは Smart Brain が使えない患者用。
体動が強い場合は Motion Free（MF） を使用。
:::

:::

## ■ 注意点

:::danger 重要
- 緊急オーダーの第一選択は Smart Brain。Rapid はSmart Brainでも対応不可の患者用。
- 体動が強い患者は Rapid より Motion Free（MF） を使用。
:::

:::warning 撮像に関する注意
- 脂肪塞栓・アミロイドアンギオパチー・microbleeds 疑い → SWIを追加撮像
- 脳外科のMRA依頼：指示コメントに CI・long・AN など指定がある場合はそれに従う
- TIA・ふらつきの依頼 → long MRA を撮像
- MRA 1chunk はアーチファクト（乱流の影響）が出現した箇所のみ撮像
:::

:::info 補足
- T1 COR TFE は必要であれば撮像。CORは脳幹に平行（橋の後縁に沿って平行ライン）
- 検査内容によっては頭部MRAではなく long MRA を撮像する
:::
