# 頭部MRI 動脈瘤・椎骨動脈解離

最終改訂：2025年11月4日

## ■ 検査概要

脳動脈瘤・椎骨動脈解離を評価するための検査。
MRAでのWillis動脈輪の描出と、解離評価のためのBPAS撮像が特徴。

## ■ 断面設定

### 動脈瘤 MRA（Head AN MRA）

Willis動脈輪の範囲を撮像（動脈瘤撮像のため、範囲が狭い）
MRAカット：内頚動脈のみ残す（外頚や枝はできる限りカット）

![](assets/images/head/aneurysm/an_mra_range.png)

:::info 撮像範囲の目安
- 下：SagでC5の後ろに走行する内頸動脈の部位をギリギリ含める
- 上：CorでM3を全て含めるのが理想だが、M3の中央あたりまで含んでいればOK
- 上記はあくまでも目安。動脈瘤がある部位を必ず含めること
:::

### VA解離 BPAS COR

解離がある血管に平行に撮像する

![](assets/images/head/aneurysm/bpas_cor_range.png)

## ■ 注意点

:::warning 動脈瘤
- 頭部MRAは Willis動脈輪の範囲を撮像（動脈瘤のため範囲が狭い）
- MRAカット：内頚動脈のみ残す（外頚や枝はできる限りカット）
- T2 DRIVE は指示があれば動脈瘤のある範囲のみに絞って撮像
:::

:::warning VA解離
- 頭部MRAは椎骨動脈を少し広めの範囲に含める
- 頭部MRAではなく long MRA の指示がある場合もある
- BPAS COR は解離がある血管に平行に撮像
:::
