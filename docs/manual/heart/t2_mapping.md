# T2マッピング解析

:::info シリーズ番号
検像送信時のシリーズ番号：2
:::

---

## 1. 解析開始

1. T2map を選択（この時 Images が一番多いものを選ぶ）
2. 左上の Analysis をプルダウン
3. Cardiac MR Analysis を選択
4. Select Protocol は T2 を選択
5. Start Analysis Application を選択

---

## 2. Motion Correction

6. Reject map and go to (Re)Calculate を選択
7. Motion Correction を選択
8. Selected Map をプルダウン → Corrected を選択
9. 左上の ⇨ をクリック

---

## 3. 心筋トレース（内壁・外壁）

10. T1マッピングと同様に 内壁・外壁をトレースする
    - ※ 乳頭筋は心筋に含めずトレースすること

---

## 4. セグメント化

11. Create new spoke をクリック（セグメント数は 6）
12. セグメント化されたことを確認
13. Accept segmentation and to results を選択

---

## 5. 解析結果表示

14. 右側に解析結果が表示される

---

## 6. カラースケール設定（rainbow2）

15. 右のアイコンをクリック
16. T2 の Low を「0」、High を「120」 に変更して Apply
17. 中央のアイコンをプルダウン → rainbow2 を選択
18. ブルズアイの色が反映されたことを確認

---

## 7. 画像保存

19. 左側の心筋の中部のスライスをクリック
20. 右下の表の Slice No が 「2」 になっていることを確認
21. この画面を全画面保存（Save screen snapshot as）
22. File name に 「T2 Map」 を入力
23. 保存先は 「Local」

---

## 8. ISP での保存・送信

24. Exit をクリック → Save & Exit をクリック
25. 保存先は Local
26. 保存した画像上で 右クリック → Copy To を選択
    - ※ 他に解析を行う場合はまとめて送信すること
27. 検像を選択 → OK

---

## 9. 検像端末での操作

:::warning 転送保留を忘れずに
送信したら必ず 転送保留 してください！
既に原画像が送信されているので、「一部転送済」「未転送」「転送済」を選択しないと該当患者が検像端末に表示されない場合があります。
:::

28. 転送保留した後、解析結果上で 右クリック → タグ編集
29. シリーズ番号に 「2」 を入力
30. 保存 → はい → 送信
