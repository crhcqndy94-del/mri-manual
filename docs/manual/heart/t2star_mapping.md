# T2*マッピング解析

:::info シリーズ番号
検像送信時のシリーズ番号：**3**
:::

---

## 1. 解析開始

1. **T2\*map** を選択（この時 Images が一番多いものを選ぶ）
2. 左上の **Analysis** をプルダウン
3. **Cardiac MR Analysis** を選択
4. Select Protocol は **T2\*** を選択

---

## ※ Protocol に T2* がない場合

5. 画面上部の **⇦** をクリック
6. 上段を選択 → **Scan Type を T2\*** に変更
7. 下段も同じく **T2\*** に変更
8. 画面上部の **⇨** をクリック
9. Select Protocol は **T2\*** を選択
10. **Mapping** を選択

---

## 2. 解析開始（通常手順）

11. **Start Analysis Application** を選択

---

## 3. Motion Correction

12. **Reject map and go to (Re)Calculate** を選択
13. **Motion Correction** を選択
14. **Selected Map** をプルダウン → **Corrected** を選択
15. 左上の **⇨** をクリック

---

## 4. 心筋トレース（内壁・外壁）

16. T1マッピングと同様に **内壁・外壁をトレース**する
    - ※ **乳頭筋は心筋に含めずトレース**すること

---

## 5. セグメント化

17. **Create new spoke** をクリック（セグメント数は **6**）
18. セグメント化されたことを確認
19. **Accept segmentation and to results** を選択

---

## 6. 解析結果表示

20. 右側に解析結果が表示される

---

## 7. カラースケール設定（rainbow2）

21. 右のアイコンをクリック
22. **T2\* の Low を「0」、High を「80」** に変更して **Apply**
23. 中央のアイコンをプルダウン → **rainbow2** を選択
24. ブルズアイの色が反映されたことを確認

---

## 8. 画像保存

25. 左側の心筋の中部のスライスをクリック
26. 右下の表の Slice No が **「2」** になっていることを確認
27. この画面を全画面保存（**Save screen snapshot as**）
28. File name に **「T2Star Map」** を入力
29. 保存先は **「Local」**

---

## 9. ISP での保存・送信

30. **Exit** をクリック → **Save & Exit** をクリック
31. 保存先は **Local**
32. 保存した画像上で 右クリック → **Copy To** を選択
    - ※ 他に解析を行う場合はまとめて送信すること
33. 検像を選択 → **OK**

---

## 10. 検像端末での操作

:::warning 転送保留を忘れずに
送信したら必ず **転送保留** してください！
既に原画像が送信されているので、「一部転送済」「未転送」「転送済」を選択しないと該当患者が検像端末に表示されない場合があります。
:::

34. 転送保留した後、解析結果上で **右クリック → タグ編集**
35. シリーズ番号に **「3」** を入力
36. **保存** → **はい** → 送信
