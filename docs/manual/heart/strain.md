# ストレイン解析（院内手順）

:::info シリーズ番号
検像送信時のシリーズ番号：6
:::

---

## 1. 解析開始

1. CINE Sag・CINE Sax・CINE 4CH を選択し、Strain をクリック
2. Strain をクリック

---

## 2. 画像の割り当て

3. CINE Sag・Sax・4CH をそれぞれ ドラッグ&ドロップ
4. Automatically segment にチェックが入っている状態で 「Next」 をクリック

---

## 3. セグメンテーション処理

5. 「Process」 をクリック
6. 「Next」 をクリック

---

## 4. セグメントモード選択・レポート保存

7. Segments Mode から 「AHA-17 Segments Model」 を選択
8. Save Report をクリックしてレポートを保存

---

## 5. 解析結果の表示・Key Image 設定

9. 患者一覧に戻り、該当患者を 右クリック → View
10. 解析結果をダブルクリックで表示
    - ※ 表示後、4画面表示から最大化して 1画面表示 に変更すること

11. 1枚目から順番に 右クリック → Mark Image as Key Image（合計 5枚）
    - 1枚目
    - 2枚目
    - 3枚目
    - 4枚目
    - 5枚目

12. Exit
13. Yes

---

## 6. ISP での送信

14. 保存した5枚の画像を選択 → Copy to
    - ※ 他に解析を行う場合はまとめて送信すること
15. 送信先は KENZO

---

## 7. 検像端末での操作

:::warning 転送保留を忘れずに
送信したら必ず 転送保留 してください！
既に原画像が送信されているので、「一部転送済」「未転送」「転送済」を選択しないと該当患者が検像端末に表示されない場合があります。

ストレイン解析のモダリティは『OT』 になるので、並べ替えの時は注意してください。
:::

16. 転送保留した後、解析結果上で 右クリック → タグ編集
17. シリーズ番号に 「6」 を入力
18. 保存 → はい → 送信
