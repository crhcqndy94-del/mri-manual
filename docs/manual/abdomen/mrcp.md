# MRCP（胆嚢・膵臓）

最終改訂：2025年11月4日

## ■ 検査概要

胆嚢・胆管・膵臓の評価。食事確認必須（絶食が望ましい）。

:::warning
**上腹部（MRCP・胆嚢・膵臓）検査：食べていたら検査の待ち時間を説明。待てないなら予約日変更。**
:::
