:::info 撮像範囲の考え方

- 3D：リンパ節も見ているため、枚数を増やして広めに撮像
- 舌目的のTRA：撮像範囲を舌に絞る
- 甲状腺目的のTRA：撮像範囲を甲状腺に絞る
- 舌の時は舌を動かさないよう患者に説明
:::
