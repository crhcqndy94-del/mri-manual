# 着替え・問診・MRI室入室手順

最終改訂：2025年11月13日

## ■ 検査概要

全MRI検査に共通する入室前フローチャート。
患者を更衣室まで案内し、以下の順で確認する。

---

## ■ 入室前チェックフロー

### STEP 1 — 患者確認

1. 患者に名乗ってもらう（名前を確認）
2. 問診表を確認
   - 問診表がある場合 → 日付・名前・内容を確認
   - ない場合 → 検査の説明を聞いているか確認
     - 聞いている → 問診表を印刷
     - 聞いていない → 外来へ戻る

### STEP 2 — 食事確認

| 検査 | 制限 | 対応 |
|------|------|------|
| 上腹部（MRCP・胆嚢・膵臓） | 検査開始4時間前から絶食 | 時間が空いていなければ待ち時間を説明。待てないなら予約日変更 |
| 造影検査 | 検査開始4時間前から絶食 | 時間が空いていなければ待ち時間を説明。待てないなら予約日変更 |
| それ以外 | 原則不問 | — |

### STEP 3 — 基本問診（フローチャート参照）

| 確認項目 | 対応 |
|---------|------|
| 頭部の手術歴の有無 | ある場合はフローチャート確認 |
| 心臓の手術歴の有無 | ある場合はフローチャート確認 |
| その他手術歴の有無 | 確認 |
| 入れ歯・補聴器 | 外す |
| インプラント・歯科矯正 | 材質・MRI可否確認。不明なら歯科に確認。確認不可なら本人了承で検査可 |
| カラーコンタクト | 外す |
| コンタクト（通常） | ケース・保存液を確認。ない場合は売店で購入 |
| シップ・カイロ | 外す |
| ニトロダーム・ニコチネル | 外す |
| ボディアート | 火傷の可能性を説明 |
| ヘアピン・ピアス・時計・指輪 | 外す |
| マスカラ（ラメあり） | 化粧を落とす |
| マニキュア・ネイル | 落としてもらう（オフにしないと検査不可） |
| 糖尿病など測定医療器具 | 主治医に相談 |
| 妊娠の有無 | 主治医に説明を聞いているか確認 |
| 磁石式バルーン挿入患者 | バルーンキャップに付け替えたら検査可 |
| 乳房術後患者 | エキスパンダー挿入していたら検査不可 |

:::danger
乳房術後患者でエキスパンダーが挿入されている場合は検査不可
:::

---

## ■ 着替え

問診に問題がなければ着替えを行う。

| 衣服 | 対応 |
|------|------|
| ヒートテック | 脱衣する |
| エアリズム | 脱衣しない |
| 金属が装着されている衣服 | 全て脱衣 |

肌着1枚・下着1枚で、検査着に着替える

---

## ■ 前投薬が必要な患者

対象：心疾患・緑内障・前立腺肥大・糖尿病

- フローチャートを確認
- 注射後から1時間程度の待ち時間を説明
- ボースデルありの検査：軟便になりやすいことを説明

![](assets/images/checkin/premedication_flow.png)

---

## ■ 金属チェック・入室

1. 着替え終了後、もう一度問診を行う
2. 前を向いてもらい身体全体をチェッカーで確認
3. 後ろを向いてもらい身体全体をチェッカーで確認
4. 更衣室のロッカーの鍵を確認（鍵は検査室にかけ、患者に説明）
5. 問題がなければMRI室へ入室

---

## ■ 造影患者の追加問診

| 確認項目 | 対応 |
|---------|------|
| アレルギー | 何のアレルギーか確認 |
| 喘息 | 症状を聞く |
| 造影の同意書 | 日付・名前・内容を確認。なければ説明を聞いているか確認 |
| 食事の有無 | 食べていたら待ち時間を説明 |
| 体重 | およその体重を聞く |
| アルコール消毒 | 確認する |
