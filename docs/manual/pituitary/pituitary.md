# 下垂体MRI（単純・造影）

最終改訂：2025年11月4日

## ■ 検査概要

下垂体腫瘍・低身長・ホルモン異常などに対する検査。
造影はDynamic（インジェクター使用・同時スタート）。

- 下垂体Dynamic造影の場合は**Surveyから全てのシーケンスを 19.MainServer へ送信**

:::
