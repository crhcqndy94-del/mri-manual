# MRIカンファレンス議事録 2026年2月

開催日：2026年2月5日

---

## 1. IMRTの着替え・運用

- 専用固定具2段重ね使用（MR室内固定具の上に患者個人固定具）
- 着替えは治療科で実施してから来室
- 問診はこれまで通り

## 2. 医療安全

- ペン類・ノック式マジック・カイロの持ち込みが連続2回発生（技師ボールペンは現状維持）
- 移動時は手足を内側に入れる
- マーカー貼付時のテープかぶれ確認・かぶれあれば透明テープ使用

## 3. 小児科鎮静時立ち合い

- 固定具使用時は小児科医立ち合い必須
- 固定具なしは不要

## 4. 造影剤アレルギーの患者プロファイル記載

- 副作用発生後に放射線科医師にアレルギー副作用として登録確認

## 5. カルテの「＃」への記載

- 技師と看護師の両方記載でも二重記載にならない

## 6. 膝関節ルーチン変更

- T2*/Sag → PD FS/Tra

## 7. 当日予約枠開放

- 予約枠が埋まらない日が多いため開放（小林Dr承諾済み）
- 当日緊急 = 電話連絡必要
- 当日予約枠 = 電話連絡不要

## 8. 3.0T膀胱・前立腺DWI

- ノイズ再調整済みシーケンス導入
