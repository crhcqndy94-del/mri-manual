# MRIカンファレンス議事録 2025年5月

開催日：2025年5月23日

---

## 1. 同意書・問診票変更（マグネットネイル）

- 全種類ネイル不可に変更
- 注意喚起文書配布予定
- 旧版混在1年間（旧版では明確に確認できた場合のみ実施）
- 緊急撮像は主治医責任で実施コメント必須
- 新追加文言：
  - ①金属・磁性装飾品の安全未確認時は中止
  - ②埋め込み後は1週間前に主治医相談

## 2. Feet-first・Head-first

- Head-firstをデフォルト
- 四肢はFeet-first固定
- インジェクタ位置は固定

## 3. T1 in/out Subtractionの有無

- 腎臓は必須（自動作成）
- MRCPや肝臓は不要
- それ以外は臨機応変に対応

## 4. 車椅子運用

- 操作室で預かり継続
- MR室扉を必ず閉める徹底

## 5. 責任者報告

- 担当技師は16:15終業
- 私語注意・節度ある勤務態度
- 名前ダブルチェック徹底
- 患者接遇クレームへの注意
- 頭部造影MRAは初回でも基本不要
- 看護師人員配置への口出し禁止
